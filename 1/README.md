（任務一）以【迴圈】產生指定費式數列的所有因數組合
輸入：一個正整數 N
步驟：以【迴圈】產生從 1 開始的費氏數列前 N 個數字，再以【迴圈】找出每個數字差值最
小的兩個因數，以其乘積表示該數字並予以編號。若費氏數已超過長整數容許範圍，
就停止產生並輸出對應的訊息。
輸出：在螢幕上依序顯示費氏數列前 N 個數字，為每個數字附上對應的兩個因數相乘，並註
明執行過程中外層和內層迴圈執行次數。輸入格式和輸出訊息必須和範例程式一致，
內層迴圈執行次數可以不同。


（任務二）以【遞迴】產生指定費式數列的所有因數組合
輸入：同任務一。
步驟：同任務一，只是將【迴圈】改為【遞迴】。以【遞迴】產生從 1 開始的費氏數列前 N
個數字，再以【遞迴】找出...並予以編號。
輸出：同任務一，只是將迴圈執行次數改為執行過程中外層和內層遞迴呼叫次數。輸入格式
和輸出訊息必須和範例程式一致，內層遞迴執行次數可以不同。


（任務三）以【遞迴】產生指定長度的所有【組合】
輸入：一個不大於 10 的正整數 M。
步驟：從 0, 1, 2, ..., 9 挑出 M 個相異的數字來產生各種【組合】，逐一予以編號[1], [2], [3], ...，
並測量整體的執行時間。
輸出：在螢幕顯示 M 個相異數字對應的各種【組合】，最後註明整體執行時間，單位是毫秒。
輸入格式和輸出訊息必須和範例程式一致，測量到的執行時間可以不同。
